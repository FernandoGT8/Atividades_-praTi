# 🚀 Repositório de Atividades - Formação Full Stack (+praTi)

Este repositório concentra as entregas práticas e exercícios de código desenvolvidos durante a formação intensiva em **Desenvolvimento Full Stack Júnior** pela +praTi e Codifica. 

O objetivo deste repositório é registrar a evolução técnica contínua, partindo da estruturação de algoritmos e lógica de programação pura até a construção de interfaces web e, futuramente, APIs estruturadas no Back-end.

## 🛠️ Tecnologias Utilizadas
Nesta etapa inicial do projeto, as seguintes tecnologias foram aplicadas:

*   **JavaScript (ES6+)**: Lógica de programação, manipulação de arrays/objetos e interação com a DOM.
*   **HTML5**: Estruturação semântica de páginas web.
*   **CSS3**: Estilização, responsividade e layout (Flexbox/Grid).
*   **Git & GitHub**: Versionamento de código e boas práticas de commits.

---

## 📂 Estrutura do Repositório

O projeto está dividido modularmente conforme o avanço das disciplinas:

### 📁 Módulo 01: Lógica de Programação e Algoritmos
Foco na resolução de problemas matemáticos e lógicos utilizando JavaScript de forma isolada (Node.js/Console). 
*   `Atividade01_Modulo01.js`
*   `Atividade02_Módulo01.js`
*   `Atividade03_Módulo01.js`

### 📁 Módulo 02: Desenvolvimento Web (Front-end Essencial)
Construção de uma aplicação web estática e funcional, demonstrando navegação entre páginas, separação de responsabilidades e manipulação de formulários.
*   `index.html`: Landing page principal.
*   `catalogo.html`: Página de listagem/produtos.
*   `login.html`: Estrutura de autenticação de usuário.
*   `style.css`: Folha de estilos centralizada.
*   `script.js`: Arquivo de scripts para interatividade da interface.
